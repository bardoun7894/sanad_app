### CHAT SUBSCRIPTION ($5/month)

**Cost**: $5/month (auto-renews with card, manual with bank)

**What's Included**:
```

âœ… UNLIMITED messaging/chat
âœ… 24/7 message access
âœ… Chat with AI assistant
âœ… Message history
âœ… Notifications for replies
âœ… Can @ mention in community

âŒ Voice/video calls

```

**Payment Methods**:

**Option 1: Credit Card (Instant)**
- Provider: PayPal or 2Checkout
- Payment: Visa, Mastercard
- Process: Instant (seconds)
- Activation: Immediate
- Renewal: Auto-monthly (PayPal handles)
- Cancellation: User cancels via PayPal
- Fees: 2.9% (PayPal) or 4.3% (2Checkout)
- Expected: 80% of users

**Option 2: Bank Transfer (Manual)**
- Provider: Wise or Payzone/CMI
- Process: User transfers $5 from their bank
- Reference: Unique code (REF-USER123-202512)
- Verification: Admin checks in messages (1-24h)
- Activation: Admin clicks "Verify"
- Renewal: Manual (user initiates transfer again)
- Fees: 0.5-1% (very low)
- Expected: 20% of users

**Target Users**: Active users wanting daily support

**Revenue**: Primary (40% of user base)

**What Happens**:
```

3. Picks date & time (therapist's availability)
4. Selects duration
```

**Target Users**: Serious users needing professional help

**Revenue**: Secondary (10% of user base)
